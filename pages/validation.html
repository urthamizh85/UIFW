Note: The above message setting is default only, you can change the messages.

Submit Validation
$("#someformid").formValidation("validate");

Note: The above line will return true of false

Field Validation
$("#somefieldid").formValidation("validateField");

Note: The above line will return true of false

Remove error
$("#somefieldid").formValidation("removeError");

Note: The above line will remove mentioned field error only.

Show custom error message
$("#somefieldid").formValidate("showCustomError", {
msg : "Your custom message",
where : $("#someErrorShowId") //! Here you can refer where you want the error will display.
});

Note: The above line will remove mentioned field error only.
