Fullscreen

Simple wrapper for cross-browser usage of the JavaScript Fullscreen API, which lets you bring the page or any element into fullscreen. Smoothens out the browser implementation differences, so you don't have too.
Disclaimer : IE10 and less are not supported. Safari doesn't support use of the keyboard in fullscreen.

Full Screen the page

Javascript :

$('[data-fullscreen]').click(function () { if (screenfull.enabled) { screenfull.request(); } });

Fullscreen an element

Click on the image to make it fullscreen.

Javascript :

$('img').click(function () { if (screenfull.enabled) { screenfull.toggle(this); } });
