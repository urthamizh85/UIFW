CNSI Core UI

scrollMove(top, duration)

Scrolls the page to a specified top position

top (number) : (Top position of the page)

duration (number in milliseconds) : (duration for animation)

Example

myApp.scrollMove(0, 500);

scroll(scrollId, options)

Custom styled browser scroll

scrollId (string|jquery selector) : Element's id to which scroll is applied

options (object{}) : contentHeight (Maximum height for the content)

Example

myApp.scroll("#pro_lang_scroll", {
   contentHeight : "200px"
});

formreset(options)

Method to reset a form

options (object{}) : formId (id of the form)

Example

myApp.formreset({
   formId : "#settings_pin_form"
});

widgetCollapse(options)

Method for Collapsible widget

options (object{}) : iconName (icons for the widget header), callback (include callback functions)

Example

myApp.widgetCollapse({
   iconName : {
     up : "fa-chevron-up",
     down : "fa-chevron-down"
   },
   callback : function() {
     // some callback function
   }
});

loadPage(obj)

Load a page through ajax call

obj (object{}) : throbber (throbber will open), url (url of the page to be loaded)), data (can send some data), datatype (Type of data to be sent)

Example

myApp.loadPage({
   throbber : {
     throbberText : "Searching..."
   },
   url : myApp.root + loc_url,
   data : { data1 : "some data"},
   datatype : "json",
   success : function(param) {
     //your code here
   },
   fail : function(param1,param2,param3) {
     //your code here
   },always: function(param1,param2,param3) {
     //your code here
   }
});

throbber(obj)

Throbber window

obj (object{})

Example

myApp.throbber({
   throbberText : 'Please wait...',          // Text to show
   waitTime : 3000                           // After this time continue message will be shown
   hideTime : 3000                           // Throbber will autohide after this time
   headerText : 'Throbber Title',            // Title for the Throbber
   multiCallback : {                         // For multiple callback functions
     counter : 3,                            // No. of callback functions
     duration : [2000, 2000, 2000],          // duration for each callback function
     callback : [throbberMsg1, throbberMsg2, throbberMsg3]  // callback functions as array
   },
});

notification(obj)

Notification Alerts

obj (object{})

Example

myApp.notification({
   "id" : $("myId"),                            // Unique Notification Id
   message : "Your notification message!"       // Message to be displayed
   duration : 20000,                            // After this time Notification will autohide
   status : "info"|"danger"|"success"|"warning" // different status color
});

matchHeight(id1, id2)

Match the height between two elements and return the maximun height

id1 : Element1 Id

id2 : Element2 Id

Example

myApp.matchHeight($("#nav_left"),$("#page"))

dateConvertion(obj)

Converts date string to date object

obj (object{})

Example

myApp.dateConvertion({
   data : [{
     "start" : "10,8,2014,5,23,00",
     "end" : "10,8,2014,23,23,00",
   }, {
     "start" : "10,01,2014,10,32,00",
     "end" : "10,01,2014,10,36,00",
   }],
   "key" : "start" || "end"
});

dateConvertion(obj)

Converts an Object to Array

obj (object{})

Example

myApp.objectToArray({
   "11" : "Acupuncturist",
   "15" : "Alzheimer Center (Dementia Center)",
   "16" : "Ambulance",
   "17" : "Art Therapist",
   "26" : "Adult Companion",
   "48" : "Air Carrier"
});

tryCatch(trycode, callback)

A simple tryCatch method to catch error

trycode is a function

Example

myApp.tryCatch(function(){
     //try some of your code
},function(){
     //then apply here.
});

apply(arguments, data)

Automatic method execution

Note: If arguments have any function , it will automatically applied with data if available.

Example

myApp.apply(arguments);

setPageHeight()

Sets height for the page

Note: This method depends on matchHeight().

Example

myApp.setPageHeight();

mediaScreen()

Return the type of device based on the screen resolution

Example

myApp.mediaScreen();

searchForm(formId, callback)

Apply callback functions while searching

formId : Id of the form

callback : function to be applied

Example

myApp.searchForm('#search_form', callback);

stopPropagate()

Generic method for jQuery stopPropagate()

This functionality will apply for elements having attribute data-propagate='stop'

Example

myApp.stopPropagate();

customClick()

Elements having attribute data-click can have functions as value for this attribute. While click this elemet, specified function will execute

Example

myApp.customClick();

currentTime(options)

Returns curren time (13:32:50)|(1:32 PM) based on the configuration

Param (options) is object{}

Example

myApp.currentTime({
   join : ":",
   am : " AM",
   pm : " PM",
   AMPM : true,
   seconds : false
});

formatTime12h(time)

Returns formatted time (hh:mm (AM|PM))

Param (time) is string

Example

myApp.formatTime12h("13:32:56");      //Result : "01:32 AM"

zeroFormat(val)

Returns (val) begin with 0 if (val) is below 10

Param (val) is number

Example

myApp.zeroFormat(7);      //Result : 07

customSerialize(obj, avoidkey, joiner)

Returns the form elements name and corresponding values as an object

Param (obj) is object{}

Example

myApp.customSerialize($('#form_validate').serializeArray())

objectextend(options)

Returns overwritten object{}

Param (options) is object{}

Example

myApp.objectextend({
   from : {
     id : "some",
     width : 100,
     height : 200,
     background : "#fff"
   },
   to : {
     id : "other",
     width : 200,
     height : 300,
     left : 0,
     top : 10
   }
});

Result Description : The to:{} data overwritten by from:{} data

Note : if from.key == to.key matched then only it will ovewrite,else key and value will append.

timeOut(options)

Execute a function after a specified time

Param (options) is object{}

Example

myApp.timeOut({
   duration : 1000, // default duration is 1 , 1000 = 1 seconds
   callback : function() {
     //your code here
   }
});

countDown(params)

Execute a specified no. of methods in the given interval

Param (params) is object{}

Example

var newContDown = new myApp.countDown({
   counter : 2,                // How many time you want run counter
   duration : [3000, 3000],    // Each counter duration
   callback : [method1, method2]  // Functions as array
});
newContDown.init(newContDown.setting); // Count down initiation

pageFontSize()

Increase or Minimize the text size of the page

Example

myApp.pageFontSize();

setLeftMenuPosition()

Leftmenu position is controlled by this method based on the Screen Resolution. For tablet, only icons will be visible. For mobile, leftnav will be hidden.

Usage

myApp.setLeftMenuPosition();

collapseSideMenu()

Leftmenu show/hide functionality is done by this method.

Usage

myApp.collapseSideMenu();

setLeftMenuActive()

This method will change the corresponding leftmenu to active state.

Usage

myApp.setLeftMenuActive();

loadPageFromNavigation()

Ajax loading of content while clicking leftmenu is donw by this method.

Usage

myApp.loadPageFromNavigation();

openRightNavigation()

This method will open the Rightnav panel

Usage

myApp.openRightNavigation();

autoPageLoad()

Initial ajax loading of content is done by this method.

Usage

myApp.autoPageLoad();
